Spotify Music Trends Analytics Project

--------------------------------------------------------------------------------------------------------------------------------------------------------------------
This repository contains the code and documentation for the Spotify Music Trends Analytics project. The goal of this project is to analyze Spotify tracks, artists, and albums to uncover insights regarding popularity trends, genre distribution, and artist performance using a comprehensive data analytics pipeline.

The project follows a standard data analytics workflow:

Data Cleaning & Preprocessing:

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Using Python (Pandas, NumPy, Matplotlib, Seaborn) to clean the raw dataset, handle missing values, remove duplicates, standardize formats, and create new features.

Database Storage & Querying:

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Utilizing SQL (MySQL) to structure the cleaned data and perform analytical queries.

Visualization & Reporting: 

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Creating an interactive dashboard in Power BI for visual exploration of key music trends.

Analytical Queries:

----------------------------------------------------------------------------------------------------------------------------------------------------
The following types of queries were executed to derive insights:

Total Tracks: 

--------SELECT COUNT(*) AS total_tracks FROM spotify_tracks;

Top Artists by Track Count: 

--------SELECT artist_name, COUNT(*) AS track_count FROM spotify_tracks GROUP BY artist_name ORDER BY track_count DESC LIMIT 10;

Top Tracks by Popularity: 

-----------SELECT track_name, track_popularity, artist_name FROM spotify_tracks ORDER BY track_popularity DESC LIMIT 10;

Release Trends: 

----------SELECT release_year, COUNT(*) AS track_count FROM spotify_tracks GROUP BY release_year ORDER BY release_year;
